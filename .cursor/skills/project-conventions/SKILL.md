# Соглашения: API и размеры

Каноничные документы — **источник истины**; здесь только навигация и TL;DR.

## API (маршруты, URL, формы отчётности)

Полное описание: **`docs/api-conventions.md`** (в корне репозитория).

**Кратко:**

- Версия в URL: `/api/v1/*`.
- Формы отчётности: `/api/v1/report-{номер}/...` (например `report-6406`).
- Пути без завершающего слеша для коллекций и ресурсов.
- Коллекции — множественное число, kebab-case для составных сегментов, ID в параметрах пути.

Перед изменением или добавлением endpoint открой `docs/api-conventions.md` и проверь разделы по именованию и примерам.

## Размеры файлов и объёмы (байты)

Полное описание: **`docs/conventions/file-size-conventions.md`**.

**TL;DR из документа:**

| Где | Правило |
|-----|--------|
| БД | `bigint`, значения в байтах |
| API | числа в байтах |
| UI / человекочитаемо | форматирование или поля `*Human` |
| Имена | `*Bytes`, `*Size` для байт; `*Human` для строк |

При полях «размер файла», «объём», «storage» — сверяйся с `file-size-conventions.md`.
