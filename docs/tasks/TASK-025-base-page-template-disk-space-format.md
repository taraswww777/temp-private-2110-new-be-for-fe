# TASK-025: BasePageTemplate — отображение diskSpaces из сырых данных BE, утилита formatByteSize

**Статус**: planned  
**Ветка**: —

---

## Краткое описание

В **BasePageTemplate** изменить логику отображения `diskSpaces`: принимать сырые данные с BE (размеры в байтах), выводить строку вида «Свободно {free} из {total}» с человекочитаемыми значениями и единицами измерения. Ввести/использовать утилиту форматирования размера (например, `formatByteSize`), пересмотреть полезность `packageDtoToPacketDto`, `parseSizeToMB`, `storageVolumeToDiskSpace`.

---

## Цели

1. **BasePageTemplate** — изменить логику отображения `diskSpaces`:
   - на вход ожидаются сырые данные с BE (в т.ч. размеры в байтах);
   - строка вывода: «Свободно {free} из {total}», где `free` и `total` — отформатированные строки с единицей измерения (например, «512 Мб», «1.25 Гб»).

2. **Утилита форматирования размера**:
   - преобразование размера из байтов в строку с подходящей единицей измерения;
   - точность: максимум 2 знака после запятой;
   - в качестве основы подойдёт логика/имя, связанные с `storageVolumeToDiskSpace`, с переименованием в нечто вроде **`formatByteSize`** (байты → строка с единицей).

3. **Пересмотр утилит** — оценить необходимость и использование:
   - `packageDtoToPacketDto`;
   - `parseSizeToMB`;
   - `storageVolumeToDiskSpace`  
   и при необходимости упростить/объединить/переименовать/удалить.

---

## Обязательные требования (к выполнению)

- **BasePageTemplate**: отображение «Свободно {free} из {total}», где `free` и `total` — результат утилиты форматирования (строка с единицей измерения), на входе — данные с BE (размеры в байтах).
- **Утилита** (например, `formatByteSize`): байты → строка с единицей измерения, не более 2 знаков после запятой.
- Пересмотр и при необходимости рефакторинг/удаление: `packageDtoToPacketDto`, `parseSizeToMB`, `storageVolumeToDiskSpace`.
