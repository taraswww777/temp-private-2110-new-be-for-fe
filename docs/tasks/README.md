# Управление заданиями проекта

Этот документ описывает, как работать с заданиями при помощи AI-ассистента (Cursor Agent).

---

## 🤖 Работа с AI-ассистентом

AI-ассистент автоматически управляет заданиями согласно строгому протоколу. Вам нужно только давать команды и принимать результаты работы.

### Как создать задание

Просто опишите AI-ассистенту, что нужно сделать:

```
Создай задание: Добавить API для экспорта данных в Excel
```

AI-ассистент автоматически:
- Создаст файл задания с уникальным номером
- Заполнит все необходимые разделы
- Добавит запись в манифест
- Установит статус "Бэклог"

### Как начать работу над заданием

```
Начнём работу над TASK-XXX
```

AI-ассистент **обязательно** спросит подтверждение, затем:
- Создаст git-ветку
- Обновит статус на "В работе"
- Начнёт выполнение согласно плану

### Как принять выполненную работу

После завершения работы AI-ассистент **обязательно** спросит:

```
Задание TASK-XXX выполнено.

Проверьте, пожалуйста, критерии приёмки:
- [список критериев]

Принимаете работу?
```

**Варианты ответа:**

✅ **Принимаю работу:**
```
Да / Принимаю / Хорошо
```
AI обновит статус на "Выполнено" и зафиксирует дату завершения.

❌ **Нужны доработки:**
```
Нет, нужно доработать: [описание что нужно исправить]
```
AI продолжит работу и снова запросит приёмку.

### Как отменить задание

```
Отмени задание TASK-XXX
```

AI спросит причину отмены и зафиксирует её в документации.

---

## 📋 Статусы заданий

| Иконка | Название | Описание |
|--------|----------|----------|
| 📋 | Бэклог | Задание создано и ожидает начала работы |
| ⏳ | В работе | Задание выполняется AI-ассистентом |
| ✅ | Выполнено | Задание завершено и принято вами |
| ❌ | Отменено | Задание отменено и не будет выполнено |

---

## 📂 Структура файлов

```
docs/tasks/
├── README.md              # Этот файл
├── tasks-manifest.json    # Список всех заданий (автоматически управляется AI)
├── TASK-001-название.md   # Файл задания
├── TASK-002-название.md
└── TASK-XXX-название.md
```

### Манифест заданий

Файл `tasks-manifest.json` содержит список всех заданий проекта. AI-ассистент автоматически обновляет его при изменении статусов.

**Вы можете просматривать манифест**, чтобы увидеть:
- Все задания проекта
- Текущие статусы
- Даты создания и завершения

---

## ✍️ Произвольные задания

Если вы **самостоятельно создали файл задания** (написали вручную), AI-ассистент может его "причесать" и добавить в систему.

### Как добавить произвольное задание

1. Создайте файл задания в любом формате
2. Попросите AI:

```
Причеши задание из файла custom-task.md и добавь его в систему
```

AI-ассистент:
- Прочитает ваш файл
- Приведёт его к стандартному формату
- Присвоит номер TASK-XXX
- Добавит в манифест
- Создаст корректный файл задания
- **Автоматически удалит исходный файл**

**Важно:** После "причёсывания" AI создаст новый файл с правильным именем (`TASK-XXX-название.md`) и автоматически удалит ваш исходный файл.

## 🔍 Просмотр заданий

### Все задания

Откройте файл `tasks-manifest.json` — он содержит полный список.

### Фильтрация по статусу

В манифесте найдите задания с нужным статусом:

- **Бэклог**: `"status": "backlog"`
- **В работе**: `"status": "in-progress"`
- **Выполнено**: `"status": "completed"`
- **Отменено**: `"status": "cancelled"`

### Просмотр деталей задания

Откройте файл задания `TASK-XXX-название.md` — он содержит:
- Описание задачи
- Цели
- Критерии приёмки
- План выполнения
- Историю изменений

## 🎯 Гарантии протокола

AI-ассистент работает по строгому протоколу и **ГАРАНТИРУЕТ**:

✅ **Задание не будет завершено без вашего подтверждения**
- AI всегда спросит о приёмке работы
- Статус "Выполнено" устанавливается только после вашего "Да"

✅ **Работа не начнётся без вашего разрешения**
- AI спросит перед созданием ветки
- Статус "В работе" устанавливается только после вашего подтверждения

✅ **Все изменения документируются**
- Каждое изменение статуса фиксируется
- Уточнения сохраняются в файле задания
- Манифест всегда синхронизирован

✅ **Коммиты имеют правильный формат**
- Все коммиты начинаются с `TASK-XXX`
- Описания коммитов на русском языке

**Вы:**
```
Да, начинаем

```
Да, принимаю
```

```

📁 Манифест обновлён
