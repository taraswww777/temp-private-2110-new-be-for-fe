# Шаблоны промптов

Здесь хранятся шаблоны промптов для начала работы над задачами. Использование шаблонов позволяет передавать ассистенту только **динамические данные** (номера задач, ветки), не повторяя полный текст каждый раз.

## Как использовать

1. Открой нужный шаблон (например, `start-task-with-subrepo.md`).
2. Подставь переменные в своём сообщении или укажи их явно.
3. Можно написать коротко: **«Выполнить задачу TASK-032. В подрепо: ветка `feature/VTB-533/FE-Dokumentaciya-dlya-mokov`, задача VTB-533»** — ассистент подставит значения в шаблон согласно `.cursorrules`.

## Переменные шаблона «Начало задачи с подрепозиторием»

| Переменная    | Пример       | Описание |
|---------------|--------------|----------|
| `TASK_ID`     | TASK-032     | Номер задачи в основной репе (документ в `docs/tasks/`) |
| `VTB_ID`      | VTB-533      | Номер задачи в подрепо (префикс коммитов в temp-private-2110) |
| `BRANCH_NAME` | feature/VTB-533/FE-Dokumentaciya-dlya-mokov | Имя ветки в подрепо temp-private-2110 |
